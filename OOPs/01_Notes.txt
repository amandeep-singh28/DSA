->Constructor Working in Inheritance
(i)Java internally called parent class everytime even if we don't write super().
(ii)If there are only default constructor present in the both the classes, than first parent class default constructor will run and than child class default constructor.
(iii)super() must be the first statement inside the default constructor of the child class.
(iv)super() must not be written twice.
(v)If the parent class is having parameterized constructor than we need to specify super() in the child class constructor and also in the parameters inside the super() keyword
(vi)In case of default constructor super() is automatically called.

Example:-

class Student {
    Student() { // Default Constructor
        System.out.println("Parent class");
    }
}

class Hostel extends Student {
    Hostel() { // Default Constructor
        // super()
        System.out.println("Child class");
    }
}

public class constructorWorking {
    public static void main(String[] args) {
        Hostel h1 = new Hostel();
    }
}

Example:-

package OOPs;

class Student {
    Student(int x) {
        System.out.println("Parent class");
    }
}

class Hostel extends Student {
    Hostel(String a) {
        super(10);
        System.out.println("Child class");
    }
}

public class constructorWorking {
    public static void main(String[] args) {
        Hostel h1 = new Hostel("Hello");       
    }
}


->Method overriding and Method overloading
(i)Method overriding
-Same function name and same parameters.
-Cannot be done in the same class.
-It only happens when class inherits something(Inheritance).

(ii)Method overloading
-Same function name and different parameters.
-Can be done in the same class.


->Uses of super()
(i)It can access the overriden methods
Example:-

class ABC {
    void display() {
        System.out.println("ABC class");
    }
}

class XYZ extends ABC {
    void display() {
        super.display();
        System.out.println("XYZ class");
    }
}

public class useOfSuper {
    public static void main(String[] args) {
        XYZ obj1 = new XYZ();
        obj1.display();
    }
}

(ii)It can access public data members of the parent class

Example:-

class ABC {
    String name = "Amandeep Singh";
    void display() {
        System.out.println("ABC class");
    }
}

class XYZ extends ABC {
    String name = "Amandeep";
    void display() {
        super.display();
        System.out.println(name);
        System.out.println(super.name);
    }
}

public class useOfSuper {
    public static void main(String[] args) {
        XYZ obj1 = new XYZ();
        obj1.display();
    }
}

(iii)It can call the default constructor of the parent class.
-super() must be the first statement inside the child class default constructor

Example:-

class ABC {
    ABC() {
        System.out.println("ABC Class");
    }
}

class XYZ extends ABC {
    XYZ() {
        super();
        System.out.println("XYZ Class");
    }
}

public class useOfSuper {
    public static void main(String[] args) {
        XYZ obj1 = new XYZ();
    }
}

(iv)To access the parameterized constructor of the parent class

Example:-

class ABC {
    ABC() {
        System.out.println("ABC Class");
    }
    ABC(String a) {
        System.out.println("Name:" + a);
    }
}

class XYZ extends ABC {
    XYZ() {
        super("Amandeep");
        System.out.println("XYZ Class");
    }
}

public class useOfSuper {
    public static void main(String[] args) {
        XYZ obj1 = new XYZ();
    }
}
